# UI Styles

This folder contains UI styling assets like colors, fonts, and sprite assets.

## What Goes Here

### Color Palette
Define consistent colors for the game:
- **Primary**: #2C3E50 (Dark blue-gray)
- **Secondary**: #3498DB (Bright blue)
- **Success**: #2ECC71 (Green)
- **Warning**: #F39C12 (Orange)
- **Danger**: #E74C3C (Red)
- **Background**: #ECF0F1 (Light gray)
- **Text**: #2C3E50 (Dark)
- **Disabled**: #95A5A6 (Gray)

### Font Assets
- Main UI font (TMP Font Asset)
- Header font (TMP Font Asset)
- Number font (TMP Font Asset)

### UI Sprites
- Button backgrounds
- Panel backgrounds
- Icons (see Icons folder)
- Progress bars
- Dividers

### Style Sheets
- Button styles (normal, hover, pressed, disabled)
- Text styles (headers, body, labels, values)
- Panel styles (borders, backgrounds)

## Usage

Apply these styles consistently across all UI to maintain visual coherence.
