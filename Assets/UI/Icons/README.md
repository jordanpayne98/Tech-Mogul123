# UI Icons

This folder contains icon sprites for UI elements.

## What Goes Here

### System Icons
- Time control icons (Pause, Play, Fast Forward)
- Navigation icons (Employees, Products, Contracts, Market)
- Status icons (Warning, Error, Success, Info)

### Role Icons
- Developer icon
- Designer icon
- Marketer icon
- Hybrid role indicators

### Stat Icons
- Money/cash icon
- Morale indicator
- Burnout indicator
- Skill icons (Dev, Design, Marketing)
- Quality star/rating icons

### Action Icons
- Hire/add icon
- Fire/remove icon
- Edit/settings icon
- Details/info icon
- Close/cancel icon

### Product/Contract Icons
- Software product icon
- Hardware product icon
- Service product icon
- Contract difficulty indicators
- Progress/completion icons

## Icon Specifications

### Size
- Base size: 64x64 pixels
- Export as sprite sheets or individual files
- Include @2x versions for high-DPI

### Style
- Flat design
- Consistent line weight
- Match color palette
- Corporate/professional aesthetic

### Format
- PNG with transparency
- SVG source files (optional)
